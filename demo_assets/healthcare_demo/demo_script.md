# Five-Minute Demo Script

## 1. Local-First Setup

Show the header and explain that transcripts, outputs, audit logs, and bundles stay local.

## 2. Skill Pack

Load `skill_pack.json`. Point out that the researcher controls roles, concepts, disfluencies, nonverbals, and selected metrics without changing code.

## 3. Single Transcript

Paste or upload `participant_001.txt`. Run local analysis and show:

- Base Metrics
- Question Type Metrics
- Care Plan Commitment Metrics
- Concept Count Metrics

## 4. Study Batch

Use the README batch block in the Study Workspace panel. Run the batch and show:

- 3 successful runs
- aggregate JSON path
- CSV exports

## 5. Agentic Extension

Save a plugin request, queue a job, show the generated runbook, and record evidence.

## 6. V6 Platform Signals

Show the bundle manifest, audit endpoint, approved library endpoint, and secure-offline profile endpoint from the API if there is time.
